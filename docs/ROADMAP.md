# 股票交易训练沙盘开发规划

## 当前状态

项目已经具备一个可用的训练台雏形：

- 本地网页训练台：K 线、成交量、下单、持仓、成交、复盘。
- 市场引擎：结构化模拟行情、手续费、印花税、滑点、T+1。
- 历史盲测：支持导入 `data/historical/*.csv`，匿名股票代码和日期。
- 图表复盘：买卖点 B/S 标记、十字光标、资产曲线。
- 工程基础：单元测试、README、GitHub 远程仓库。

## 产品目标

这个项目不做普通模拟盘，而是做“股票交易训练系统”。它应当帮助使用者在可加速、可复盘、可重复的市场环境中训练交易纪律、仓位控制、入场理由和离场执行。

## 优先级路线

### P0：训练闭环质量

1. 交易计划/交易日志
   - 下单前记录买卖理由。
   - 记录止损计划、目标价、观察点。
   - 成交后把计划绑定到成交记录和 K 线标记。

2. 每笔交易复盘
   - 卖出后统计该笔交易盈亏。
   - 记录买入后最大浮亏、最大浮盈。
   - 展示是否按计划离场。
   - 当前已完成基础版本：按 FIFO 配对买入/卖出，展示盈亏、持有 K 线数、最大浮盈/浮亏、止损/目标触碰。

3. 训练记录保存
   - 每轮训练结束后保存收益率、最大回撤、胜率、盈亏比、交易次数。
   - 支持查看历史训练结果。
   - 当前已完成基础版本：手动保存训练摘要到 `data/training_runs.jsonl`，页面展示最近记录。

### P1：图表和决策辅助

1. 均线系统
   - MA5、MA10、MA20、MA60。
   - 成交量均线。
   - 当前已完成基础版本：K 线图支持 MA5、MA10、MA20、MA60 开关和悬浮均线值。

2. 技术指标
   - MACD、KDJ、RSI。
   - 默认收起，避免页面过载。

3. 图表交互
   - 缩放、拖拽、左右回看。
   - 固定十字光标。
   - 买卖点连线和盈亏标记。

### P2：真实市场细节

1. A 股规则增强
   - 涨跌停限制。
   - ST 股票涨跌幅。
   - 停牌。
   - 价格最小变动单位。

2. 数据源增强
   - 批量 CSV 管理。
   - 可选接入 AkShare、Tushare 或 BaoStock。
   - 数据质量检查和缺失 K 线提示。

### P3：训练任务

1. 专项训练模式
   - 熊市少亏训练。
   - 趋势突破训练。
   - 止损执行训练。
   - 仓位控制训练。

2. 个人成长曲线
   - 多轮训练成绩趋势。
   - 常见错误标签。
   - 交易纪律评分。

## 自评审

### 暂缓项

- 暂缓直接接外部在线数据源：网络、依赖、接口权限不稳定，先把本地 CSV 和训练闭环做扎实。
- 暂缓复杂指标堆叠：指标过多会让页面像看盘软件，而不是训练系统。
- 暂缓排行榜：当前阶段更需要个人复盘和成长曲线。

### 当前最值得先做

第一阶段选择“交易计划/交易日志”。

理由：

- 它直接服务训练目标，不只是增加视觉功能。
- 实现范围可控：订单模型、服务端接口、前端下单面板和成交展示。
- 后续每笔交易复盘、纪律评分、交易教练都依赖这份计划数据。

## 第一阶段执行项

1. 订单模型增加计划字段：
   - `reason`
   - `stop_loss`
   - `target_price`
   - `review_note`

2. 成交记录绑定计划字段：
   - 成交面板展示交易理由。
   - K 线悬浮到买卖点时展示计划摘要。

3. 前端下单面板增加计划输入：
   - 买卖理由。
   - 止损/目标。
   - 备注。

4. 测试：
   - 订单计划能随成交保存。
   - 服务端 JSON 能返回计划字段。
   - 前端 JS 语法通过。
