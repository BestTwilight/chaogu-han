# 历史行情 CSV 放置目录

把真实股票历史行情 CSV 放在这个目录下，启动网页训练台后选择“历史盲测”，再点击“重置”即可训练。

每个 CSV 至少需要这些字段：

```csv
date,open,high,low,close,volume
2023-01-03,10.20,10.50,10.10,10.32,1234567
2023-01-04,10.32,10.80,10.22,10.76,1456789
```

可选字段：

```csv
symbol,industry
```

说明：

- `date` 支持 `YYYY-MM-DD`、`YYYY/MM/DD`、`YYYYMMDD`。
- 系统会隐藏真实股票代码和真实日期，显示为 `STOCK_A`、`STOCK_B`。
- 默认历史盲测每轮抽取 180 根 K 线。
- 文件名可以用股票代码，例如 `600519.csv`、`000001.csv`。
